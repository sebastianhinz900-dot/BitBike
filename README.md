# 🚲 BITBike – Fahrrad & E-Bike Werkstatt-App

Eine vollständige, offline-fähige Werkstatt-Management-App für Fahrrad- und E-Bike-Service-Betriebe. Läuft vollständig im Browser – keine Installation, kein Server, keine Cloud erforderlich.

![BITBike](https://img.shields.io/badge/version-2.0-orange) ![License](https://img.shields.io/badge/license-MIT-blue) ![HTML](https://img.shields.io/badge/HTML-Single%20File-green)

## 🚀 Schnellstart

Einfach index.html im Browser öffnen – fertig. Keine Installation notwendig.

GitHub Pages: https://sebastianhinz900-dot.github.io/BitBike

## ✨ Features

- Dashboard: Umsatz, Termine heute, offene Aufträge, Lager-Warnungen
- Terminplanung: Kalender, WhatsApp-Integration
- Reparaturaufträge: Positionen, Arbeitswerte, Lager-Buchung, PDF-Druck
- Kundenverwaltung: Kartei, Fotos, KI-Analyse, Reparaturhistorie
- Lagerverwaltung: Bestandswarnungen, EK/VK-Preise
- EÜR: Einnahmen-Überschuss-Rechnung
- Rechnungen: PDF-Export, 19% MwSt / Kleinunternehmer §19
- Shimano EV: Explosionszeichnungen direkt suchen
- Software Updates: Firmware-Datenbank für E-Bike-Hersteller
- KI-Assistent: Groq und Gemini, Sprachsteuerung, Bildanalyse
- 5 Themes: Standard, Dark Tech, Neon, Warm, Midnight
- Supabase Sync: optional, geräteübergreifend

## 🛠️ Technologie

Einzel-Datei-App (HTML + CSS + JS) – keine Build-Tools, keine Abhängigkeiten.

## 📦 Installation

git clone https://github.com/sebastianhinz900-dot/BitBike.git
Dann index.html im Browser öffnen – oder GitHub Pages nutzen.

## ⚙️ KI-Assistent

1. Einstellungen → KI-Assistent → Groq wählen
2. Kostenloser Key: https://console.groq.com/keys
3. Key eintragen → fertig
